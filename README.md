# online-course

Interactive course materials for **R Fundamentals for Health Research** (CHPR/EPI 202), built with [Quarto](https://quarto.org) and [WebR](https://docs.r-wasm.org/webr/latest/).

- **Website:** https://r-pophealth-research.github.io/online-course
- **Source:** `.Rmd` files in `tutorials/`, `hw/`, and `final/`
- **WebR site:** `.qmd` files are generated by `scripts/prepare_webr_docs.R`

## Local development

```bash
Rscript scripts/prepare_webr_docs.R
quarto render
quarto preview
```

Open `online-course.Rproj` in RStudio to work with the `.Rmd` source files directly.
